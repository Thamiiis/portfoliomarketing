Este foi um trabalho muito marcante pois foi realizado utilzando apenas Inteligência Artificial, sendo um pré-requisito para a matéria Inteligência Artificial, da Fatec Carapicuíba.

Para a criação desse vídeo foi utilizado diversas ferramentas de IA

Chat GPT: para a criação das ideias;

Copy.Ai: para a criação dos textos;

Fliki: para a criação das imagens a partir das descrições que o Copy.Ai e o ChatGPT forneceram. Além disso, foi utilizado para a criação do vídeo como um todo, inclusive para a voz e a música.
